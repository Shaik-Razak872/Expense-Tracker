# 💰 Expense Tracker

A modern **Expense Tracking Web Application** that helps users manage daily spending, track expenses, and analyze financial patterns with interactive charts.

---

## 🚀 Features

* Add, edit, and delete expenses
* Categorize expenses (Food, Travel, Shopping, etc.)
* Interactive charts and dashboards
* Real-time expense tracking
* Responsive and modern UI

---

## 🛠️ Technologies Used

* **React.js**
* **Vite**
* **JavaScript**
* **CSS**
* **Chart Libraries**

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/Shaik-Razak872/Expense-Tracker.git
```

Install dependencies:

```
npm install
```

Run the project:

```
npm run dev
```
